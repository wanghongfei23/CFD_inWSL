详细描述

区块求解器类

该类负责管理CFD计算的主要流程，包括初始化、时间推进、输出等核心功能。 它集成了网格读写、方程求解、边界条件处理等功能模块。

◆ stepsLoopCFL()

void BlockSolver::stepsLoopCFL ( )

基于CFL条件的时间步循环

基于CFL条件控制时间步长的求解循环

此函数实现了基于CFL条件的自适应时间步长求解循环。在每次迭代中：

  1. 检查是否需要输出当前结果
  2. 计算满足CFL条件的最大时间步长
  3. 根据输出间隔调整时间步长
  4. 执行一次时间步进求解
  5. 更新时间和步数计数器
  6. 输出当前求解状态信息 循环持续到达到设定的结束步数为止。
注解
该方法使用可变时间步长，确保数值稳定性
参见
getTimeIntervalExplicit() 获取满足CFL条件的时间步长
solve() 执行单步求解
outputGrid() 输出网格信息
outputPrim() 输出原始变量
