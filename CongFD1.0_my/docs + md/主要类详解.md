# 主要类详解

## 1. Info 类
Info 类是整个程序的信息管理中心，存储所有的配置参数和运行时状态。它包含了：

具体成员变量：
- InterMethod spMethod：空间方法，默认为WCNS5
- EquationType eqType：方程类型，默认为EULER
- int nCase：算例编号，默认为2
- real t：当前时间，默认为0
- real dt：时间步长，默认为0.0001
- int step：当前步数，默认为0
- int endStep：结束步数，默认为25000
- int dim：问题维度，默认为1
- int outputInterval：输出间隔，默认为100
- real outputDt：输出时间间隔，默认为0.01
- real outputT：输出时间计数器，默认为0
- real CFL：CFL数，默认为0.5
- bool fixedtimeSteps：固定时间步标志，默认为true
- real implicitCFL：隐式CFL数，默认为0.01
- real maxImplicitStep：最大隐式步数，默认为100
- DiffMethod diffMethod：差分方法，默认为TRAD6
- InterMethod interMethod：插值方法，默认为WCNS5
- SourceType sourceType：源项类型，默认为SOURCENULL
- std::array<int,3> iMax：网格最大索引，默认为{201,201,2}
- std::array<double,6> calZone：计算区域，默认为{0,0.3,0,0.3,0,2}

## 2. Block 类
Block 类代表计算域中的一个网格块，包含：

具体成员变量：
- Data coorVer：顶点坐标数据
- Data coorCel：单元中心坐标数据
- Data intervalCel：单元间距数据
- int nVer：顶点数量
- int nCel：单元数量
- bool inited：初始化标志
- int dim：网格维度
- std::array<int,3> iMax：网格节点最大索引
- std::array<int,3> icMax：内部单元最大索引

## 3. Data 类
Data 类是一个通用的数据容器，用于存储各种物理量：

具体成员变量：
- std::vector<real> data：实际数据存储
- int n：网格点数，默认为200
- int nVar：变量数，默认为1
- std::vector<std::string> varName：变量名称列表

## 4. Equation 类
Equation 类处理不同物理方程的数值计算：

具体成员变量：
- bool inited：初始化标志，默认为false
- int n：网格点数
- int nCons：守恒变量数
- int nPrim：原始变量数
- int dim：问题维度
- Data* cons：守恒变量数据指针
- Data* rhs：右手端数据指针
- Data* prim：原始变量数据指针
- EquationType type：方程类型

## 5. BlockSolver 类
BlockSolver 是主求解器类，负责协调整个计算过程：

具体成员变量：
- CgnsIO cgnsIO：CGNS输入输出对象
- Info* info：信息对象指针
- Block* block：网格块指针
- Initializer* initer：初始化器指针
- Equation* eqn：方程对象指针
- Bnds* bnds：边界条件指针
- SpDistributor* spDis：空间分布器指针
- Data* cons：守恒变量数据指针
- Data* rhs：右手端数据指针
- SourceTerm* sourceTerm：源项指针
- TimeMethod timeMethod：时间方法，默认为RK3SSP
- long timesteps：时间步计数器，默认为0

## 6. Initializer 类
Initializer 类负责初始化整个计算环境：

具体成员变量：
- Info* info：信息对象指针

## 7. SpDistributor 类
SpDistributor（空间分布器）类负责将空间离散任务分发到各个计算维度：

具体成员变量：
- int nCons：守恒变量数
- int nPrim：原始变量数
- int dim：问题维度
- std::array<int,3> iMax：各方向最大索引
- Data* prim：原始变量数据指针
- Data* cons：守恒变量数据指针
- Data* rhs：右手端数据指针
- Bnds* bnds：边界条件管理器指针
- Info* info：Info对象指针

## 8. SpaceDis 类
SpaceDis（空间离散）类实现具体的空间离散操作：

具体成员变量：
- Data* data：数据指针
- Data* rhs：右手端数据指针
- Info* info：Info对象指针
- std::shared_ptr<OneDBnd> bndL,bndR：左右边界指针
- long long timep：计时变量
- int n,nVar,nPrim：网格点数、变量数等
- int idim：维度索引
- int i0,offset：起始索引和偏移量
- int nHalf：半网格点数
- std::array<real,3> norm：法向量
- std::shared_ptr<Data> flux_d：通量数据指针
- std::shared_ptr<OneDBnd> fBndL,fBndR：通量边界指针
- EquationType fluxType：通量类型
- DiffMethod diffMethod：差分方法
- InterMethod interMethod：插值方法

## 9. Reconstructor 类
Reconstructor 类是重构算法的基类：

具体成员变量：
- std::shared_ptr<Data> valsR：重构后数据指针
- Data* vals：原始数据指针
- int i0：起始索引
- int offset：偏移量
- std::shared_ptr<OneDBnd> bndL,bndR：左右边界指针
- Info* info：Info对象指针
- int n：网格点数
- int nval：变量数

## 10. Bnds 类
Bnds 类管理边界条件：

具体成员变量：
- int dim：问题维度
- std::array<int,3> iMax：各方向最大索引
- std::vector<std::shared_ptr<OneDBnd>> oneDBnds：一维边界条件向量
